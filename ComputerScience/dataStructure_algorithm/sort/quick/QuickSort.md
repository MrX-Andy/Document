###### 快速排序

> 基本描述

```
小于等于基准数的  放在 左边子集； 大于基准数的 放在右边子集；
left = 1  --> size-1;
right = size-1    -->    1;
left <= right;
pivot = array[0]; 
x = 0;
初始：
85, 24, 63, 45, 17, 31, 96, 50
pivot = 85, left = 0, right = 7
i = left = 0, j = right = 7;

85, 24, 63, 45, 17, 31, 96, 50
50, 24, 63, 45, 17, 31, 96, 50
50, 24, 63, 45, 17, 31, 85, 96
```
