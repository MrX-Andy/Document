###### 归并排序

> 典型的 分治的思想  divide-and-conquer

> 分
```
8, 4, 9, 5, 7, 1, 3, 6, 2
分 9/2 = 4
再分 4/2 = 2
再分 2/2 = 1
[8]  [4] [9]  [5]  [7]  [1]  [3]  [6]  [2]
```
> 和
```
合 1*2 = 2
[4, 8]  [5, 9]  [7, 1]  [3, 6]  [2]
再合2*2 = 4
[4, 5, 8, 9]  [1, 3, 6, 7]  [2]
再合
[1, 3, 4, 5, 6, 7, 9]  [2]
再合
[1, 2, 3, 4, 5, 6, 7, 8, 9]
```
